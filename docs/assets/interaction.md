# 才赋生涯®招商网页交互设计

## 核心交互功能

### 1. 收益测算器（核心交互）
**功能描述：** 区县合伙人收益互动测算工具
- 左侧：参数设置面板
  - 投资金额：固定5万元（不可修改）
  - 预计第一年成交客户数：滑动条（5-50个）
  - 预计裂变客户数：滑动条（每老客裂变1-5个新客户）
  - 客单价：固定19800元
  
- 右侧：实时收益展示
  - 第一年收益测算：回本阶段（前5单50%分成）+ 盈利阶段（后续35%分成）
  - 第二年收益测算：所有客户35%分成 + 裂变客户收益
  - 投资回报图表：柱状图展示两年收益对比
  - 关键数据：回本周期、总收益、ROI

- 底部：行动召唤
  - "立即成为合伙人"按钮 → 弹出联系表单

### 2. 多维度痛点分析交互
**功能描述：** 7个维度的市场需求可视化展示
- 7个标签页切换：国家、社会、学校、家庭、学生、行业、从业者
- 每个标签页包含：
  - 痛点描述卡片（3-5个核心痛点）
  - 数据支撑（市场规模、政策要求等）
  - 相关图片/图标
- 点击标签页时，内容区域平滑过渡切换

### 3. 产品体系对比展示
**功能描述：** 51-985体系 vs 传统服务的对比
- 左右分栏对比布局
- 传统服务（左侧）：
  - 单一学科补习
  - 临时抱佛脚
  - 唯分数论
  - 头痛医头
  
- 才赋51-985（右侧）：
  - 全周期陪伴
  - 一站式解决方案
  - 情绪价值+精细化服务
  - 标本兼治
- 点击"了解更多"展开详细服务内容和频次

### 4. 专家团队展示轮播
**功能描述：** 核心专家团队展示
- 自动轮播：李军老师、谢丽萍老师、颜晴老师
- 每位专家包含：
  - 头像照片
  - 职位头衔
  - 核心成就（如"18年行业深耕"、"5000+学子成功案例"）
  - 专业领域
- 左右箭头手动切换
- 底部指示点显示当前位置

## 交互反馈设计

### 微交互
- 按钮悬停效果：颜色渐变 + 阴影扩展
- 卡片悬停：轻微上浮 + 阴影加深
- 数字滚动动画：页面加载时数字从0递增到目标值
- 进度条动画：收益测算时动态填充

### 页面切换
- 平滑滚动导航
- 标签页切换淡入淡出
- 轮播图滑动过渡

### 表单交互
- 联系表单：实时验证 + 成功提示
- 滑动条：实时数值显示
- 下拉选择：平滑展开动画

## 响应式设计
- 移动端：收益测算器改为上下布局
- 平板端：保持桌面端布局，适当缩小间距
- 手机端：导航折叠为汉堡菜单

## 转化路径设计
1. **认知阶段**：Hero区域 → 市场痛点 → 品牌权威
2. **兴趣阶段**：产品服务 → 专家团队 → 成功案例
3. **决策阶段**：合作模式 → 收益测算 → 保障机制
4. **行动阶段**：联系表单 → 电话咨询 → 预约面谈

每个阶段都设置明确的CTA按钮，引导用户进入下一步。
